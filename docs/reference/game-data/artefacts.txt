Select Artefact

Artifacts are a standard mechanic in the Heroes of Might and Magic series. You find an artifact on the map or receive it as a battle reward — it appears in the hero's inventory.

What are artifacts and how they work

When hovering over an artifact in the inventory, you will see all the information about it. It always consists of the following points:

Rarity

Shows how rare the artifact is. The higher the rarity — the harder it is to obtain and the stronger the bonuses. In the game there are4 rarity categories:

Common

Rare

Epic

Legendary

Slot

Equipment slot on the hero: head, neck, torso, hands, legs, rings, misc items, etc. On the right side of the site there is a filter with all slots — convenient for searching.

There areunique slots— only one artifact can be equipped in them:

head

neck

And there aremultiple identical slots, for example:

rings — 2

misc items — 4

As soon as the artifact is equipped in a free slot of the hero — it immediately starts providing its bonuses.

Effect

Bonuses provided by the artifact while it is equipped.This can be:

increase to primary stats

combat effects (for example +1 Speed to friendly creatures)

enhancement of one or several hero skills

Upgraded

Upgrading the main bonus of the artifact using alchemical dust.

Where to get it — [described here].

Set

Which set the artifact belongs to. Details about sets — [here]. In the list you can click on an artifact and see what it provides.

Set Effects

Additional effects that appear when a certain number of artifacts from the same set are equipped on the hero.

How to collect artifacts:

Here you can obtain common and rare artifacts:

Raiders’ Camp

Black Tower

Ancient Crypt

Here you can obtain legendary and epic artifacts:

Twilight Bloom

Research Laboratory

Dragon Utopia

Alternative sources

Build the Artifact Merchant in your towns

Search for artifacts directly on the map
