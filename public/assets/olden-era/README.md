# Olden Era Wiki Assets

Downloaded from the Heroes of Might and Magic: Olden Era Official Wiki:

https://wiki.hoodedhorse.com/Heroes_of_Might_and_Magic_Olden_Era/

The wiki footer states that content is available under Creative Commons Attribution-ShareAlike 4.0 unless otherwise noted. Check the individual source file pages in `manifest.json` before shipping redistributed builds or using these outside local app prototyping.

`manifest.json` records each asset's category, local public path, source file page, direct wiki image URL, and downloaded byte size.
